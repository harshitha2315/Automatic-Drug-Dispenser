# AI-Enabled Automatic Drug Dispenser

## Overview
An AI-enabled automatic drug dispenser designed to improve medication safety and adherence for elderly and dependent patients.

The system verifies the user before dispensing medicine and confirms whether the medication has been taken using sensor-based monitoring.

## Features
- Camera-based user verification
- Scheduled medicine dispensing
- PIR sensor-based intake detection
- Weight sensor-based pill confirmation
- Alert/message to next of kin if medicine is not taken
- Emergency spare key mechanism

## Technologies Used
- Embedded Systems
- IoT Concepts
- Camera Module
- PIR Sensor
- Weight Sensor
- Alert System
- Hardware Integration
